# 🚀 InstiWeb by Javier Soler Cantó

## Start project:

### ⚙️ Backend

- Execute 

```
npm install
```

- Execute

```
docker-compose up
```

- The backend will be listening on port 8080


### 🖥 Frontend

- Execute 

```
npm install
```

- Execute

```
npm start
```

- The frontend will be listening on port 5400
